== Frequently Asked Questions ==

= How do I install MT Addons plugin? =

There are two ways you can install EMT Addons plugin.

You can install the plugin from your WordPress dashboard:
1. Navigate to your WordPress Dashboard -> Plugins -> Add New
2. Search for MT Addons plugin
3. Install and Activate MT Addons from your Plugins page.

Or you can install it directly from WordPress.org
1. Locate and download MT Addons on wordpress.org
2. Upload the mt-addons-for-elementor directory to your â€˜/wp-content/plugins/â€™ directory using your preferred method (ftp, sftp, scp, etcâ€¦)
3. Install and Activate MT Addons plugin from your Plugins page.

= Can I use the plugin without the Elementor Page Builder? =

No â€“ MT Addons for Elementor was developed to work primarily with the Elementor Page Builder plugin. 

= How do I get support? =

If you require assistance while using the MT Addons for Elementor plugin, you can visit the <a href="https://wordpress.org/support/plugin/mt-addons-for-elementor" target="_blank">support forum</a> on WordPress.org, or you can <a href="https://modeltheme.ticksy.com" target="_blank">submit a ticket</a> and our support agents will respond to you as soon as possible.

= Do I need to know how to code? =

Absolutely not. MT Addons for Elementor was designed to allow you to create & edit anything you wish - simply locate the element you want to use and drag-and-drop it to a desired section of the page. You can customize every element by solely tweaking the options found in each widget, without the need to understand or use any coding.

= Does MT Addons for Elementor plugin work with any WordPress theme? =

Yes. MT Addons for Elementor is designed to work with all WordPress themes that utilize the Elementor Page Builder.
